# ProfitFlow

Prywatne, poufne repozytorium. Nie udostępniać poza organizacją.

## Opis
ProfitFlow to wewnętrzny system do zarządzania finansami i rentownością
projektów w software house.

## Podstawy techniczne
- Node.js 22
- React + tRPC
- MySQL/TiDB

## Szybki start (dev)
1. Zainstaluj zależności: `pnpm install`
2. Ustaw zmienne środowiskowe w `.env`
3. Uruchom aplikację: `pnpm dev`
